UType
=======

歌詞をタイピングするゲーム.

必要なもの
==============
- [node.js] (http://nodejs.org/)
- [npm] (https://npmjs.org/)

実行方法
==============
1. 作業自動化ツールgruntを入れる
```
$ npm install -g grunt-cli
```

2. 必要なライブラリを自動インストール
```
$ npm install
```

3. コンパイルが自動で行われる
```
$ grunt init
```

4. http://localhost:666/にサーバーが立つ
```
$ grunt server
```
